Web Porfolio

Este trabajo consistió en la realización de una página web para usarla como porfolio y así mostrar mi trabajo y mostrarme a mí mismo al público que le pueda interesar mi trabajo. En esta explico algunos de mis trabajos de forma resumida mostrando fotografías del resultado final del estos. Esta web fue diseñada con el Adobe Xd para ver como sería esta y una vez estuvieron tomadas todas las decisiones con el Brackets mediante HTML y CSS realicé la página web.

Esta sigue en proceso de desarrollo, ya que este tipo de trabajo siempre tiene que actualizarse y evolucionar. Para poder utilizarla decidí aprovechar la plataforma GitHub para poder tener un URL para así poder compartirlo y que puedan ver mis trabajos.
